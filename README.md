# Portal GISTEC

Versão inicial do portal do grupo de pesquisa **GISTEC — Geoinformação, Território e Conservação**.

## Estrutura

- `index.html`: página principal;
- `css/style.css`: identidade visual;
- `js/main.js`: menu responsivo e navegação;
- `imagens/logo.png`: logomarca;
- `dados/`: reservado para dados geoespaciais;
- `paginas/`: reservado para páginas internas.

## Publicação

```bash
git add .
git commit -m "Atualiza portal GISTEC"
git push
```
